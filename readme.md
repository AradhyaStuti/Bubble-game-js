# Bubble Game JS

A simple **interactive web game** where players click on bubbles with digits to match a randomly generated target number. The game is built with **HTML, CSS, and JavaScript** and keeps track of score, hits, and a countdown timer.

## ğŸ® How to Play

1. Open the game in your browser.
2. A **target digit** will appear in the HIT box.
3. Click on bubbles in the panel below:
   - If the bubble's number matches the target, you score **10 points**.
   - New bubbles are generated and a new target digit appears.
4. The **timer** counts down from 30 seconds.
5. When the timer reaches zero, the game ends, and your **final score** is displayed.
